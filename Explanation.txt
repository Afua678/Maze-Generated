The algorithm that we used was Prihm's algorithm, meaning that we greedily added vertices. One 
way that we modified it was by adding random edge weights as we traversed through the nodes 
rather than having predetermined edge weights. This meant that we did not need to know the 
weights before we started the algorithm, which differs from the origin algorithm. The way 
that we found the smallest edge out of all of the edge weights was by adding them to a heap, 
which was updated using the methods in UpdatingHeap.java so it would be logn to remove nodes 
and to adjust heap. We also recorded the x and y coordinates of the node, to make it easier 
for comparisons.
